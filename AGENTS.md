# Tiny Dinos

Godot game project.

## Stack
- Godot (`project.godot` at root)
- `scenes/` for .tscn files
- `scripts/` for GDScript
- `assets/` for art/audio

## Docs
- `DESIGN.md`: game design
- `PLAYTEST.md`: playtest notes
- `PROGRESS.md`: current state and what's next

## Workflow
- Read `PROGRESS.md` first to know where things stand.
- Update `PROGRESS.md` when a feature lands.
